Developed a weapon zeroing system for the Bangladesh Army. An android application was developed which can process the target (shooting paper) image and generate the correction value. Based on that correction value the application will command the machine to zero the weapon automatically. At the same time, the sub-unit commander can see the progress of firing of his soldiers by visiting the website.

  • Developed an algorithm that follows the zeroing technique and this algorithm was used for calculating the required correction.
  • An image processing system was implemented where machine learning was applied to get accurate and precise values from the target image. Python was used for creating the image processing api.
  • Built the web application using PHP, JavaScript, MySql, HTML and CSS. The android application was built with Java.
  • Built the zeroing machine using three stepper motors, Arduino Mega, etc.
